ajax means = (Asynchronous JavaScript And XML)AJAX allows the webpage to communicate with the server 
(PHP) without refreshing the entire page.

Only the required part of the page gets updated ‚úî   You click a button / submit form
No screen flicker ‚úî                                 Data is sent in the background
Super fast ‚úî                                        Page does NOT reload
Modern UI ‚úî                                         Only required data updates
AJAX lets a web page talk to the server without reloading the page.

JSON = JavaScript Object Notation
It is a lightweight format to store and transfer data between server (PHP) and browser (JS).

How AJAX and JSON work together
Most of the time:
AJAX = sending request
JSON = data format of response
Example:
-AJAX request sent
-Server returns JSON
-JavaScript reads JSON
-Page updates
-----------work flow of ajax and api and json-------------
(‚ÄúAJAX sends request ‚Üí API receives request ‚Üí 
API returns JSON data or message ‚Üí AJAX shows data in browser‚Äù)

üîÅ Correct flow (simple)
-AJAX ‚Üí sends request (from browser)
-API / Server ‚Üí processes request
-JSON ‚Üí response format
-AJAX ‚Üí reads JSON
-Browser ‚Üí displays result
